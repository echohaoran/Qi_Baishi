生文 API

文本生成服务配置

兼容 OpenAI 接口标准。填写 API URL 与 Key，点击「获取模型」从服务端拉取可用模型，再从下拉框选择默认模型。配置加密保存在本地 Keychain。

关于白石

墨

白石 Baishi

版本 0.4.2 · build 2026.06.29

白石是一款融合传统水墨美学的 AI 图像生成工具，支持文生图、图生图与风格迁移，让创作者以数字形式延续水墨精神。

开源声明

白石遵循以下开源协议与第三方依赖：

Apache 2.0 SDXL-BaiShi
MIT Mimocode Engine
BSD-3 Diffusers
Apache 2.0 Tauri 2.1
MIT Hugging Face Diffusers
Apache 2.0 Rust / wry
