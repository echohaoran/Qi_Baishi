在线生图 API

从下拉菜单选择预设供应商 · 模型自动匹配 · 填入 API Key · 点击「测试连接」验证连通性 · 点击「保存设置」即可在三个生图页使用。

生文 API

              
文本生成服务配置


              

                
兼容 OpenAI 接口标准。填写 API URL 与 Key，点击「获取模型」从服务端拉取可用模型，再从下拉框选择默认模型。配置加密保存在本地 Keychain。

                

                  

                    
                    
                    
支持 OpenAI / Ollama / LM Studio / vLLM / Azure OpenAI 等任意 OpenAI 兼容端点

                  

                  

                    
                    
                    
本地服务（Ollama / LM Studio）通常无需 Key，留空即可

白石 Baishi

                    
开源的文生图 / 图生图 / 文生文桌面创作工具，面向自媒体创作者、营销团队与设计师，聚焦中文创作流程与本地化工作台体验。

                  

                

                

                  版本 v0.1.4
                  Apache-2.0
                  macOS / Windows

白石采用 Apache License 2.0 开源许可证发布，允许在保留许可证与版权声明的前提下进行使用、修改与再分发。

项目说明

                

                  
产品结构覆盖工作台、文生图、图生图、多图融合、灵感墙、历史作品与设置模块，强调统一的中文提示词工作流、预设管理和生成结果归档。

                  
当前原型重点放在前端交互闭环与多供应商 API 接入体验，便于验证从提示词输入、生成、收藏、回看，到设置配置的完整创作链路。
