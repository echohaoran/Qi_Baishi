# 白石 BaiShi · 水墨生图桌面应用

> 为创作者而生的桌面生图工具 · 支持在线API+本地推理 · 笔意可调 · 灵感可循

白石是一款面向自媒体创作者、营销人和平面设计师的桌面生图应用。界面采用水墨画风格（宣纸底 + 浓淡墨 + 朱砂印章 + 笔触元素），背后是完全离线的 SDXL-BaiShi-v2 模型与 Rust 推理引擎。所有数据都在你的电脑上，不上传云端。

## 1. 快速开始

### 1.1 系统要求

| 项目 | macOS | Windows |
|---|---|---|
| 最低 | macOS 12 Monterey · Apple Silicon 或 Intel | Windows 10 (1909+) · x64 |
| 推荐 | macOS 14 Sonoma · M2 及以上 · 16 GB 内存 | Windows 11 · RTX 3060 及以上 · 16 GB 内存 |
| 存储 | 12 GB 可用空间（应用 + 模型权重） | 12 GB 可用空间 |
| GPU | Apple Silicon 统一内存 / Intel Iris / 外置 eGPU | NVIDIA CUDA 或 DirectML 兼容卡 |

### 1.2 安装

#### macOS

1. 在 [白石官网下载页] 下载 `BaiShi-x.x.x-macOS-arm64.dmg`（Apple Silicon）或 `…-x64.dmg`（Intel）
2. 双击打开 DMG，将 **白石** 拖入 **应用程序** 文件夹
3. 首次打开若提示"无法验证开发者"，到 **系统设置 → 隐私与安全性**，点 **仍要打开**
4. 启动后首次运行会下载模型权重（约 6.5 GB），请保持联网，完成后可断网

#### Windows

1. 下载 `BaiShi-x.x.x-windows-x64.msi`
2. 双击运行安装器，按提示完成
3. 若 SmartScreen 拦截，点 **更多信息 → 仍要运行**
4. 首次启动同样会下载模型权重

### 1.3 首次启动

1. **登录 / 注册**：使用企业邮箱 + 密码（密码至少 8 位，含字母与数字）
2. **模型下载**：检测本地无模型时会引导下载，等待 5-15 分钟（取决于网速）
3. **进入工作台**：看到欢迎信息与最近作品，开始创作

## 2. 7 屏导览

### 2.1 工作台（首页） — `workspace.html`

启动后的默认页面。包含：
- **问候语**：基于时间（早晨/午间/夜深）的中文问候
- **快速入口**：三个大卡片直达文生图 / 图生图 / 灵感墙
- **最近作品**：本周创作的 4 张缩略图，点击进入历史
- **风格趋势 Top 5**：本周社区热门风格榜单
- **本月算力**：已用 / 总计进度条 + 升级 PRO+ 入口
- **创作计时**：本周出图数、平均耗时、收藏数

### 2.2 文生图 — `text-to-image.html`

最常用的创作屏。左侧是输入与参数，右侧是生成结果。

**左侧输入区**：
- **提示词**（必填）：用一段文字描述你想要的画面，建议 30-200 字
- **字符计数**：右下角实时显示
- **智能润色**：点 ✦ 按钮可让模型帮你补全画面细节（消耗 0 算力）
- **固定提示词**：点击片段（如 `+ 飞白山水`）可追加到输入框

**参数**：
- **风格**：选择风格预设（飞白山水 / 工笔花鸟 / 写意人物 / 敦煌重彩 / 现代水墨 / 南宋院体…）
- **画面比例**：1:1 / 16:9 / 9:16 / 4:5 / 21:9
- **迭代步数**：20-50，越高越精细但越慢
- **CFG 引导强度**：1-20，越高越贴合提示词
- **随机种子**：留空为随机；填具体数字可复现相同结果

**右侧生成区**：
- **生图**按钮：显示消耗算力，点击开始
- 生成过程中显示进度（步骤 X/30）+ 实时预览
- 完成后出现 4 张候选图，点击任一张查看大图
- **下载**、**收藏**、**以图生图**、**复制提示词**等动作

### 2.3 图生图 — `image-to-image.html`

基于参考图生成新画作。

- **上传参考图**：拖拽到左侧虚线框，或点击选择（支持 PNG/JPG/WEBP）
- **提示词**：描述你想要的笔意 / 风格变化
- **重绘强度**：0-1，越高变化越大，0.3-0.5 是常见甜点
- **before/after 对比**：生成后中部有可拖拽对比滑块
- **再次生成**：基于当前结果继续演化

### 2.4 灵感墙 — `presets.html`

24 款预置风格 + 社区精选。

- **分类筛选**：山水 / 花鸟 / 人物 / 建筑 / 现代
- **卡片预览**：每张卡片是真实出图缩略 + 风格名（飞白山水 / 烟雨江南…）
- **点击进入详情**：左侧大图 + 右侧提示词 / 比例 / 步数等元数据
- **一键套用**：详情页右上角"用此风格生图"按钮，跳转文生图屏并预填参数

### 2.5 历史作品库 — `history.html`

所有生成过的作品都在这里。

- **日期分组**：今天 / 昨天 / 本周 / 本月 / 更早
- **筛选标签**：全部 / 收藏 / 飞白山水 / 工笔花鸟…
- **网格 / 列表切换**：右上角图标切换
- **单图操作**：悬停出现 ⭐ 收藏、⤓ 下载、🗑 删除
- **批量选择**：左上角勾选模式，可批量下载/删除

### 2.6 设置 / 账户 — `settings.html`

- **账户信息**：头像、邮箱、会员等级、本月算力
- **API 配置**（可选）：如需调用第三方云端 API（如火山方舟、阿里云）补充本地算力，在此填入 endpoint + key
- **存储路径**：自定义作品 / 模型的本地保存位置
- **快捷键**：查看与自定义 ⌘K 命令面板的快捷键
- **关于**：版本号、检查更新、用户协议、隐私政策

### 2.7 登录 / 注册 — `auth.html`

启动前看到的登录页。左右分栏：
- **左侧**：水墨画 hero（飞白 + 朱砂）+ 社区数据 + 创作者引言
- **右侧**：登录 / 注册表单，企业邮箱 + 密码

注册时密码至少 8 位且包含字母与数字；注册成功自动登录。

## 3. 常用功能

### 3.1 创作基本流程

1. 在 **文生图** 输入提示词（也可加 `+ 飞白山水` 等固定提示词）
2. 选择 **风格**（可选）和 **画面比例**
3. 点 **生图**（消耗算力），等待 5-15 秒
4. 在 4 张候选图中挑一张满意的作品
5. 点 ⭐ 收藏、⤓ 下载，或点 **以图生图** 继续演化

### 3.2 复用相同结果

如果想复现一张喜欢的图：在历史作品库点开它，记下 **随机种子**（生成时显示），下次生图时填入相同的种子 + 相同的提示词 + 相同的参数，即可得到近乎一致的结果。

### 3.3 提升出图质量

- **提示词写得具体**：与其"山水画"，不如"远山含黛，江面薄雾，一叶孤舟，笔意疏朗，飞白皴法"
- **使用固定提示词**：`+ 飞白山水` 比单独写"中国风"有效
- **迭代步数 30-40**：过低粗糙，过高边际收益递减
- **CFG 7-9**：过高画面会过饱和

### 3.4 节省算力

- **草稿阶段**：用 1:1 比例 + 25 步快速验证构图
- **成图阶段**：调高步数到 35-40 + 准确比例
- **风格相似时复用预设**：不必每次重写完整提示词

### 3.5 备份与导出

- 在 **设置 → 存储路径** 查看本地文件位置
- 复制整个 `<user_data>/artworks/` 目录即可备份所有作品
- 在 **历史作品库** 可批量下载为 ZIP

## 4. 设置与账户

### 4.1 算力等级

| 等级 | 每月算力 | 单次出图上限 | 单图最高步数 |
|---|---|---|---|
| Free | 500 | 1:1 · 25 步 | 30 |
| PRO | 5,000 | 21:9 · 50 步 | 50 |
| PRO+ | 20,000 | 21:9 · 50 步 + 多图生图 | 50 |

算力月底重置，未用完不累积。

### 4.2 第三方 API（可选）

如本地算力不足，可在 **设置 → API 配置** 填入云端推理服务的 endpoint 与 API key，白石会在本地算力耗尽时自动 fallback 到云端。**默认不开启**，不填则完全离线。

### 4.3 主题切换

当前版本仅 **宣纸主题**（暖米黄底 + 墨黑 + 朱砂）。后续版本计划加入：
- **夜墨**（深夜书房 · 深墨底 + 冷月白）
- **仿古**（旧书页 · 泛黄底 + 暗朱砂）

## 6. 常见问题

### Q1：首次启动很慢？

首次启动会下载模型权重（~6.5 GB），请保持联网。下载完成后所有操作离线可用。建议把模型存放在 SSD 上。

### Q2：为什么我的电脑出图很慢？

- Apple Silicon：确保使用 arm64 版本（默认）
- Windows NVIDIA：安装最新的 CUDA 驱动与 cuDNN
- Windows AMD/Intel：开启 DirectML 加速（在设置中切换后端）
- 调低步数（25-30）和分辨率（1024×1024）

### Q3：可以商用吗？

PRO / PRO+ 用户生成的作品可用于商业用途（需遵守用户协议第 4 条）。Free 用户仅供个人学习。

### Q4：和 Midjourney / Stable Diffusion WebUI 的区别？

- **完全离线**：数据不出本机
- **桌面应用**：系统级集成（拖拽、通知、快捷键）
- **水墨画专精**：内置中国水墨画数据集微调的模型
- **结构化预设**：风格可复用、可参数化

### Q5：模型能不能换成自己的？

高级用户可在 **设置 → 高级** 指定自定义 SDXL checkpoint 路径（`.safetensors`）。注意需要与基础模型同构（SDXL-base），否则会报错。

### Q6：忘记密码？

登录页点 **忘记密码** → 输入注册邮箱 → 接收重置链接（首次使用需联网）。

### Q7：报错"磁盘空间不足"？

模型 + 作品约占用 10-15 GB。请确保系统盘至少有 12 GB 可用，或在设置中更改存储路径到其他盘符。

### Q8：如何反馈问题？

- 应用内：设置 → 关于 → 反馈问题
- 邮件：`support@BaiShi.studio`
- 加入用户群（QR 在设置页底部）

## 7. 卸载

#### macOS

1. 打开 **应用程序** 文件夹
2. 把 **白石** 拖到废纸篓
3. （可选）清理数据：删除 `~/Library/Application Support/studio.BaiShi.desktop/`

#### Windows

1. **设置 → 应用 → 已安装的应用**
2. 找到 **白石** → 卸载
3. （可选）清理数据：删除 `%APPDATA%\studio.BaiShi.desktop\`

卸载会同时移除已下载的模型权重（6.5 GB）。
